CSS NOTES

CSS Statements are made up of four main components:

  1. Declaraton: the statement as a whole with curly braces.
  2. Selector: the name of the part you want to style.
  3. Property: Properties that we want to edit or modify (width, height, background, etc
  4. Value: values that we assign to a property (number, color, url, etc.)
